# Методы и оценки инвестиционной привлекательности и эффективности градостроительных объектов

## Список сокращений и условных обозначений:

**Сокращение на иностранном (английском) языке:**

DCF – discounted cash flow (метод дисконтированных денежных потоков)

ESG – environmental, social, governance (воздействующий на окружающую среду, общественный, управленческий) 

NPV – net present value (чистая приведенная стоимость)

ROA- real options analysis (анализ реальных опционов) 

**Сокращение и условные обозначения на русском языке:**

ГО – градостроительный объект

МАИ – метод анализа иерархии

НИР – научно-исследовательская работа

ТС-1, ТС-2, ТС-3 – точка сбоя 1,2,3 (в процессе оценки)

## Цель ВКР:

Разработать теоретическую и методологическую программу исследования для научно-исследовательской работы по теме: «Методы и оценки системы инвестиционной привлекательности градостроительных объектов».

## Цель НИР-1: 

Обосновать необходимость разработать и описать теорико- методологический каркас комплексной оценки инвестиционной привлекательности градостроительных объектов.

## Задачи НИР-1:

- выявить ключевые риски и параметры градостроительных объектов;
- проанализировать существующие методы оценки и выявить методологические ограничения;
- сформулировать требования к новой методике;
- определить основные модули предполагаемой методики и взаимосвязи;
- спланировать этапы дальнейшего исследования: определить необходимые данные и анализ.

## На первом этапе НИР были сделаны следующие шаги:

1. проведено комплексное исследование методологических основ оценки инвестиционной привлекательности градостроительных объектов.
2. сформулирована и структурирована метолодогическая проблема.
3. проведение критический анализ существующих методик.
4. определён каркас нового методологического решения.
5. на основе общего анализа и каркаса сформулирована гипотеза.

### Для практического использования необходимо:

1. cформировать базу типовых корректировочных коэффициентов 
2. необходимо подобрать структурированные данные по реальным проектам
